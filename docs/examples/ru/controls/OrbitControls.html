[name]

Orbit controls позволяют камере вращаться вокруг цели.
Чтобы использовать это, как со всеми файлами в папке /examples , вам придется включить файл отдельно в вашем HTML.

Пример

[example:misc_controls_orbit misc / controls / orbit ]

var renderer = new THREE.WebGLRenderer(); renderer.setSize( window.innerWidth, window.innerHeight ); document.body.appendChild( renderer.domElement ); var scene = new THREE.Scene(); var camera = new THREE.PerspectiveCamera( 45, window.innerWidth / window.innerHeight, 1, 10000 ); var controls = new THREE.OrbitControls( camera, renderer.domElement ); //controls.update() must be called after any manual changes to the camera's transform camera.position.set( 0, 20, 100 ); controls.update(); function animate() { requestAnimationFrame( animate ); // required if controls.enableDamping or controls.autoRotate are set to true controls.update(); renderer.render( scene, camera ); }

Конструктор

[name]( [param:Camera object], [param:HTMLDOMElement domElement] )

[page:Camera object]: (required) Камера для управления. Камера не должна быть дочерней по отношению к другому объекту, если только этот объект не является самой сценой.

[page:HTMLDOMElement domElement]: (optional) Элемент HTML, используемый для слушателей событий. По умолчанию это весь документ, однако если вы хотите, чтобы элементы управления работали только с определенным элементом (например, с канвой), вы можете указать это здесь.

Свойства

[property:Boolean autoRotate]

Установите в true, чтобы автоматически вращаться вокруг цели.
Обратите внимание, что если это включено, вы должны вызвать [page:.update] () в вашем цикле анимации.

[property:Float autoRotateSpeed]

Как быстро вращаться вокруг цели, если [property:Boolean autoRotate] в true. По умолчанию 2.0, что соответствует 30 секунд за оборот при 60fps.
Обратите внимание, что если [property:Boolean autoRotate] активирован, вы должны вызвать [page:.update] () в вашем цикле анимации.

[property:Float dampingFactor]

Инерция демпфирования используется, если [property:Boolean enableDamping] установлен в true.
Обратите внимание, что для того, чтобы это работало, вы должны вызвать [page:.update] () в вашем цикле анимации.

[property:HTMLDOMElement domElement]

HTMLDOMElement используется для прослушивания мыши / touch events. This must be passed in the constructor; changing it here will not set up new event listeners. По умолчанию это the whole document.

[property:Boolean enabled]

Whether or not the controls are enabled.

[property:Boolean enableDamping]

Установите значение true, чтобы включить демпфирование (инерцию), которое можно использовать для придания ощущения веса элементам управления. По умолчанию это false.
Обратите внимание, что если это включено, вы должны вызывать [page:.update] () в вашем цикле анимации.

[property:Boolean enableKeys]

Включить или отключить использование клавиш управления.

[property:Boolean enablePan]

Eвключить или отключить панорамирование камеры. По умолчанию true.

[property:Boolean enableRotate]

Включить или отключить горизонтальное и вертикальное вращение камеры. По умолчанию это true.
Обратите внимание, что можно отключить одну ось, задав минимальные и максимальные значения [page:.minPolarAngle polar angle] или [page:.minAzimuthAngle azimuth angle] к тому же значению, что приведет к тому, что вертикальное или горизонтальное вращение будет зафиксировано на этом значении.

[property:Boolean enableZoom]

Enable или disable zooming (dollying) камеры.

[property:Float keyPanSpeed]

Как быстро панорамировать камеру при использовании клавиатуры. По умолчанию 7.0 пикселей на нажатие клавиши.

[property:Object keys]

Этот объект содержит ссылки на коды клавиш для управления панорамированием камеры. По умолчанию это 4 клавиши со стрелками. controls.keys = { LEFT: 37, //left arrow UP: 38, // up arrow RIGHT: 39, // right arrow BOTTOM: 40 // down arrow } See [link:https://developer.mozilla.org/en-US/docs/Web/API/KeyboardEvent/keyCode this page] for a full list of keycodes.

[property:Float maxAzimuthAngle]

Как далеко вы можете вращаться по горизонтали, верхний предел. Диапазон между - Math.PI до Math.PI ( или Infinity без ограничений ) и по умолчанию Infinity;

[property:Float maxDistance]

Как далеко вы можете наехать камерой ( [page:PerspectiveCamera] only ). По умолчанию это Infinity.

[property:Float maxPolarAngle]

Как далеко вы можете двигаться по вертикали, верхний предел. Range is 0 to Math.PI radians, and по умолчанию Math.PI.

[property:Float maxZoom]

Как далеко вы можете отъехать камерой ( [page:OrthographicCamera] only ). По умолчанию это Infinity.

[property:Float minAzimuthAngle]

Как далеко вы можете вращаться по горизонтали, нижний предел. Range is - Math.PI to Math.PI ( or - Infinity for no limit ) and default is - Infinity;

[property:Float minDistance]

Как далеко вы можете наезжать в ( [page:PerspectiveCamera] only ). По умолчанию это 0.

[property:Float minPolarAngle]

Как далеко вы можете двигаться по вертикали, нижний предел. Range is 0 to Math.PI radians, and по умолчанию 0.

[property:Float minZoom]

Как далеко вы можете увеличить ( [page:OrthographicCamera] only ). По умолчанию это 0.

[property:Object mouseButtons]

Этот объект содержит ссылки на действия мыши, используемые элементами управления. controls.mouseButtons = { LEFT: THREE.MOUSE.ROTATE, MIDDLE: THREE.MOUSE.DOLLY, RIGHT: THREE.MOUSE.PAN }

[property:Camera object]

Управляемая камера.

[property:Float panSpeed]

Скорость панорамирования. По умолчанию это 1.

[property:Vector3 position0]

Используется внутри [method:saveState] и [method:reset] методов.

[property:Float rotateSpeed]

Скорость вращения. По умолчанию это 1.

[property:Boolean screenSpacePanning]

Defines how the camera's position is translated when panning. If true, the camera pans in screen space. Otherwise, the camera pans in the plane orthogonal to the camera's up direction. По умолчанию это false.

[property:Vector3 target0]

Used internally by the [method:saveState] and [method:reset] methods.

[property:Vector3 target]

Точка фокусировки органов управления, вокруг которой [page:.object] крутится камера. Он может быть обновлен вручную в любой момент, чтобы изменить фокус органов управления.

[property:Object touches]

Этот объект содержит ссылки на действия касания, используемые элементами управления. controls.touches = { ONE: THREE.TOUCH.ROTATE, TWO: THREE.TOUCH.DOLLY_PAN }

[property:Float zoom0]

Used internally by the [method:saveState] and [method:reset] methods.

[property:Float zoomSpeed]

Speed of zooming / dollying. По умолчанию это 1.

Methods

[method:null dispose] ()

Remove all the event listeners.

[method:radians getAzimuthalAngle] ()

Get the current horizontal rotation, in radians.

[method:radians getPolarAngle] ()

Get the current vertical rotation, in radians.

[method:null reset] ()

Сбросьте элементы управления в их состояние с момента последнего [page:.saveState] вызова, или начальное состояние.

[method:null saveState] ()

Сохранить текущее состояние элементов управления. Это может быть позже восстановлено с [page:.reset].

[method:Boolean update] ()

Update the controls. Must be called after any manual changes to the camera's transform, or in the update loop if [page:.autoRotate] or [page:.enableDamping] are set.
