Константы WebGLRenderer

Cull Face режимы

THREE.CullFaceNone THREE.CullFaceBack THREE.CullFaceFront THREE.CullFaceFrontBack

[page:constant CullFaceNone] отключает face culling.
[page:constant CullFaceBack] отброс обратных (back) граней (по-умолчанию).
[page:constant CullFaceFront] отброс лицевых граней.
[page:constant CullFaceFrontBack] отброс и лицевых и обратных.

Направление лицевой стороны

THREE.FrontFaceDirectionCW THREE.FrontFaceDirectionCCW

[page:constant FrontFaceDirectionCW] устанавливает порядок "намотки" полигонов по часовой стрелке.
[page:constant FrontFaceDirectionCCW] устанавливает порядок "намотки" полигонов против часовой стрелки (default).

Типы теней

THREE.BasicShadowMap THREE.PCFShadowMap THREE.PCFSoftShadowMap THREE.VSMShadowMap

Они определяют свойство WebGLRenderer [page:WebGLRenderer.shadowMap.type shadowMap.type].

[page:constant BasicShadowMap]дают нефильтрованные shadow maps - самый быстрый, но самое низкое качество.
[page:constant PCFShadowMap] фильтрованные shadow maps с использованием Percentage-Closer Filtering (PCF) алгоритма (default).
[page:constant PCFSoftShadowMap] фильтрованные shadow maps с использованием Percentage-Closer Soft Shadows (PCSS) алгоритма.
[page:constant VSMShadowMap] filters shadow maps using the Variance Shadow Map (VSM) algorithm.

Tone Mapping

THREE.NoToneMapping THREE.LinearToneMapping THREE.ReinhardToneMapping THREE.Uncharted2ToneMapping THREE.CineonToneMapping THREE.ACESFilmicToneMapping

Определяют свойство WebGLRenderer [page:WebGLRenderer.toneMapping toneMapping]. Это используется для аппроксимации появления большого динамического диапазона (HDR) на носителе с низким динамическим диапазоном стандартного монитора компьютера или экрана мобильного устройства.

[page:constant NoToneMapping] отключает tone mapping.
[page:constant LinearToneMapping] по-умолчанию.

См. [example:webgl_tonemapping WebGL / tonemapping] пример.
