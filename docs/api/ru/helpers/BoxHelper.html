[name]

Вспомогательный объект для графического отображения выровненного по мировой оси bounding box вокруг объекта. Фактический bounding box обрабатывается [page:Box3], а это всего лишь визуальный помощник для отладки. Это может быть автоматически изменено с помощью [page:BoxHelper.update] метода когда объект, из которого он создан, трансформируется. Обратите внимание, что объект должен иметь [page: Geometry] или [page: BufferGeometry], чтобы он работал, поэтому он не будет работать с [page: Sprite Sprites].

пример

[example:webgl_helpers WebGL / helpers]
[example:webgl_loader_nrrd WebGL / loader / nrrd]
[example:webgl_buffergeometry_drawcalls advanced / buffergeometry / drawcalls]
var sphere = new THREE.SphereGeometry(); var object = new THREE.Mesh( sphere, new THREE.MeshBasicMaterial( 0xff0000 ) ); var box = new THREE.BoxHelper( object, 0xffff00 ); scene.add( box );

Конструктор

[name]( [param:Object3D object], [param:Color color] )

[page:Object3D object] -- (optional) объект3D, чтобы показать выровненный по мировой оси boundingbox.
[page:Color color] -- (optional) шестнадцатеричное значение, определяющее цвет блока. По умолчанию это 0xffff00.

Создает новый каркасный блок, который ограничивает переданный объект. Внутренне это использует [page:Box3.setFromObject] рассчитать размеры. Обратите внимание, что это относится к любым детям.

[method:null update]()

Обновляет геометрию помощника в соответствии с размерами объекта, включая любые дочерние элементы.. See [page:Box3.setFromObject].

[method:BoxHelper setFromObject]( [param:Object3D object] )

[page:Object3D object] - [page:Object3D] to create the helper of.

Обновляет окно каркаса для пропущенного объекта..
