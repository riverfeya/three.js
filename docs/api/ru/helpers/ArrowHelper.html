[name]

3D объект со стрелкой для визуализации направлений.

пример

[example:webgl_geometries WebGL / geometries]
[example:webgl_geometry_normals WebGL / geometry / normals]
[example:webgl_shadowmesh WebGL / shadowmesh]
var dir = new THREE.Vector3( 1, 2, 0 ); //нормализовать вектор направления (преобразовать в вектор длины 1) dir.normalize(); var origin = new THREE.Vector3( 0, 0, 0 ); var length = 1; var hex = 0xffff00; var arrowHelper = new THREE.ArrowHelper( dir, origin, length, hex ); scene.add( arrowHelper );

Constructor

[name]([param:Vector3 dir], [param:Vector3 origin], [param:Number length], [param:Number hex], [param:Number headLength], [param:Number headWidth] )

[page:Vector3 dir] -- направление от origin. Должно быть unit vector.
[page:Vector3 origin] -- Point в котором arrow начинается.
[page:Number length] -- длина arrow. По умолчанию это *1*.
[page:Number hex] -- шестнадцатеричное значение для определения цвета. По умолчанию это 0xffff00.
[page:Number headLength] -- Длина головки стрелы. По умолчанию это 0.2 * length.
[page:Number headWidth] -- Длина ширины стрелки. По умолчанию это 0.2 * headLength.

свойства

Смотрите базовый класс [page:Object3D] для общих свойств.

[property:Line line]

Содержит линейную часть arrowHelper.

[property:Mesh cone]

Содержит конусную часть arrowHelper.

методы

См. базовый класс [page:Object3D] для общих методов.

[method:null setColor]([param:Color color])

color -- Желаемый цвет.

Устанавливает цвет arrowHelper.

[method:null setLength]([param:Number length], [param:Number headLength], [param:Number headWidth])

length -- Желаемая длина.
headLength -- Длина головки стрелы.
headWidth -- Длина ширины стрелки.

Устанавливает длину arrowhelper.

[method:null setDirection]([param:Vector3 dir])

dir -- Желаемое направление. Должно быть unit vector.

Устанавливает направление arrowhelper.
