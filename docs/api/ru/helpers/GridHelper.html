[name]

GridHelper - это объект для определения сеток. Сетки - это двумерные массивы линий..

Example

var size = 10; var divisions = 10; var gridHelper = new THREE.GridHelper( size, divisions ); scene.add( gridHelper ); [example:webgl_helpers Example using various helpers]

Конструктор

[name]( [param:number size], [param:Number divisions], [param:Color colorCenterLine], [param:Color colorGrid] )

size -- Размер сетки. По умолчанию это 10.
divisions -- Количество делений по сетке. По умолчанию это 10.
colorCenterLine -- Цвет центральной линии. Это может быть [page:Color], шестнадцатеричное значение или имя цвета CSS. По умолчанию это 0x444444
colorGrid -- Цвет линий сетки. Это может быть [page:Color], шестнадцатеричное значение или имя цвета CSS. По умолчанию это 0x888888

Создает новый [name] с размером 'size' и разделить на 'divisions' сегментов на сторону. Цвета не являются обязательными.
