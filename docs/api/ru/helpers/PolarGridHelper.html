[name]

PolarGridHelper - это объект для определения полярных сеток. Сетки - это двумерные массивы линий..

Пример

var radius = 10; var radials = 16; var circles = 8; var divisions = 64; var helper = new THREE.PolarGridHelper( radius, radials, circles, divisions ); scene.add( helper ); [example:webgl_helpers Example using various helpers]

Constructor

[name]( [param:Number radius], [param:Number radials], [param:Number circles], [param:Number divisions], [param:Color color1], [param:Color color2] )

radius -- Радиус полярной сетки. Это может быть любое положительное число. По умолчанию это 10.
radials -- Количество радиальных линий. Это может быть любое положительное целое число. По умолчанию это 16.
circles -- Количество кругов. Это может быть любое положительное целое число. По умолчанию это 8.
divisions -- Количество отрезков, используемых для каждого круга. Это может быть любое положительное целое число, которое больше 3. По умолчанию 64.
color1 -- Первый цвет, используемый для элементов сетки. Это может быть [страница: цвет], шестнадцатеричное значение и CSS-Color name.По умолчанию 0x444444
color2 -- Второй цвет используется для элементов сетки. Это может быть [page:Color], шестнадцатеричное значение и имя CSS-цвета. По умолчанию это 0x888888

Создает новый [name] с радиусом 'radius' и 'radials' количеством радиалей и 'circles' количеством кругов, где каждый круг сглаживается в 'divisions' раз сегментов линий. Цвета не являются обязательными.
