[page:Object3D] → [page:Line] → [page:LineSegments] →

[name]

Вспомогательный объект для визуализации [page:Box3].

пример

var box = new THREE.Box3(); box.setFromCenterAndSize( new THREE.Vector3( 1, 1, 1 ), new THREE.Vector3( 2, 1, 3 ) ); var helper = new THREE.Box3Helper( box, 0xffff00 ); scene.add( helper );

Конструктор

[name]( [param:Box3 box], [param:Color color] )

[page:Box3 box] -- Box3, для отображения.
[page:Color color] -- (optional) цвет бокса. По умолчанию это 0xffff00.

Создает новый каркас, который представляет переданный Box3.

свойства

См. базовый класс [page:LineSegments] для общих свойств.

[property:Box3 box]

The Box3 being visualized.

Methods

См. базовый класс [page:LineSegments] для общих методов.

[method:void updateMatrixWorld]( [param:Boolean force] )

Это переопределяет метод в базовом классе [page: Object3D], так что он также обновляет поле каркаса до экстента свойства [page: Box3Helper.box .box]..
