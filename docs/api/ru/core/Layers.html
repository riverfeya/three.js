[name]

ОБъект [page:Layers] object assigns an [page:Object3D] до 1 или более из 32 слоев, пронумерованных от 0 до 31 - внутренне слои хранятся в виде [link:https://en.wikipedia.org/wiki/Mask_(computing) bit mask], и по умолчанию все Object3Ds являются членами слоя layer 0.

Это может быть использовано для контроля видимости - объект должен иметь общий слой с [page:Camera camera] быть видимым, когда эта камера view рендерится.

Все классы, которые наследуются от [page:Object3D] имеют свойство [page:Object3D.layers] , которое является экземпляром этого класса.

Примеры

[example:webgl_layers WebGL / layers]

Конструктор

[name]()

Создает новый объект Layers, с членством, изначально установленным на layer 0.

свойства

[property:Integer mask]

Битовая маска, хранящая какого из 32 layers этот слой объектов в настоящее время является членом.

Методы

[method:null disable]( [param:Integer layer] )

layer - это integer от 0 до 31.

Удалить членство в этом *layer*.

[method:null enable]( [param:Integer layer] )

layer - это integer от 0 до 31.

Add membership of this *layer*.

[method:null set]( [param:Integer layer] )

layer - это integer от 0 до 31.

Set membership to *layer*, and remove membership all other layers.

[method:Boolean test]( [param:Layers layers] )

layers - a Layers object

Возвращает true если этот и переданный объекты *layers* являются членами одного слоя.

[method:null toggle]( [param:Integer layer] )

layer - это integer от 0 до 31.

Переключить членство *layer*.

[method:null enableAll]()

Добавить членство во все слои.

[method:null disableAll]()

Удалить членство из всех слоев.
