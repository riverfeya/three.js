[name]

Uniforms это глобальные [link:https://www.opengl.org/documentation/glsl/ GLSL] переменные. Они передаются в шейдерные программы.

пример

При объявлении Uniform [page:ShaderMaterial], он объявлен значением или объектом.

uniforms: { time: { value: 1.0 }, resolution: new THREE.Uniform(new THREE.Vector2()) }

Типы Uniform

Каждая форма должна иметь *value* свойство. Тип value должен соответствовать типу uniform variable в коде GLSL как указано в спецификации примитивных GLSL типов в таблице ниже. Uniform структуры и массивы также поддерживаются. GLSL массивы примитивных типов либо должен быть указан как массив соответствующих объектов THREE или как flat array содержащий данные всех объектов. Другими словами; GLSL примитивы в массивах не должен быть представлен массивами. Это правило не применяется транзитивно. Массив *vec2* массивов, каждый длиной пять векторов, должен быть массивом массивов из пяти [page:Vector2] объектов или десяти *number*s.

Конструктор

[name]( [param:Object value] )

value -- Объект, содержащий значение для настройки uniform. Это тип должен быть одним из Uniform типов описанных выше.

Свойства

[property:Object value]

Текущее значение uniform.

методы

[method:Uniform clone]()

Возвращает клон этого uniform.
Если свойство uniform имеет значение [page:Object] с методом clone(), это используется, в противном случае значение копируется по присваиванию. Значения массива распределяются между клонированными [page:Uniform]s.

См. [example:webgldeferred_animation WebGL deferred animation] для примера использования этого метода.
