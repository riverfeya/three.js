[name]

Треугольная грань используется в [page:Geometry]. Они создаются автоматически для всех стандартных типов геометрии, однако, если вы создаете пользовательскую геометрию, вам придется создавать их вручную.

Examples

[example:svg_sandbox svg / sandbox ]

[example:misc_exporter_obj exporter / obj ]

[example:webgl_shaders_vector WebGL / shaders / vector ]

var material = new THREE.MeshStandardMaterial( { color : 0x00cc00 } ); //создать треугольную геометрию var geometry = new THREE.Geometry(); geometry.vertices.push( new THREE.Vector3( -50, -50, 0 ) ); geometry.vertices.push( new THREE.Vector3( 50, -50, 0 ) ); geometry.vertices.push( new THREE.Vector3( 50, 50, 0 ) ); //создать новую грань, используя вершины 0, 1, 2 var normal = new THREE.Vector3( 0, 1, 0 ); //optional var color = new THREE.Color( 0xffaa00 ); //optional var materialIndex = 0; //optional var face = new THREE.Face3( 0, 1, 2, normal, color, materialIndex ); //добавить грань в массив граней геометрии geometry.faces.push( face ); //нормали грани и нормали вершин могут быть вычислены автоматически, если не указано иное geometry.computeFaceNormals(); geometry.computeVertexNormals(); scene.add( new THREE.Mesh( geometry, material ) );

Constructor

[name]( [param:Integer a], [param:Integer b], [param:Integer c], [param:Vector3 normal], [param:Color color], [param:Integer materialIndex] )

a — Vertex A index.
b — Vertex B index.
c — Vertex C index.

normal — (опционально) Нормаль грани ([page:Vector3 Vector3]) или массив нормалей вершин. Если передан один вектор, это устанавливает [page:.normal], в противном случае, если в этих наборах передается массив из трех векторов [page:.vertexNormals]

color — (опционально) Цвет грани [page:Color color] или массив цветов вершин [page:Color colors]. Если передан один вектор, это устанавливает [page:.color], иначе если передан массив из трех векторов то будет установлено[page:.vertexColors]

materialIndex — (опционально) какой индекс массива материалов связать с гранью.

Свойства

[property:Integer a]

Vertex A index.

[property:Integer b]

Vertex B index.

[property:Integer c]

Vertex C index.

[property:Vector3 normal]

Face normal - вектор, показывающий направление Face3. Если рассчитывается автоматически (используя [page:Geometry.computeFaceNormals]), это нормализованное перекрестное произведение двух ребер треугольника. По умолчанию *(0, 0, 0)*.

[property:Color color]

Face color - для него будет использоваться материал [page:Material.vertexColors vertexColors] свойство должно быть установлено в [page:Materials THREE.FaceColors].

[property:Array vertexNormals]

Array of 3 [page:Vector3 vertex normals].

[property:Array vertexColors]

Массив из 3-х вершинных цветов - для них будет использоваться свойство материал [page:Material.vertexColors vertexColors] должен быть установлен в [page:Materials THREE.VertexColors].

[property:Integer materialIndex]

Material index (указывает на индекс в связанном массиве материалов). По умолчанию *0*.

Methods
