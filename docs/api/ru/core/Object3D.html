[name]

Это базовый класс для большинства объектов в three.js и предоставляет набор свойств и методов для управления объектами в трехмерном пространстве.

Обратите внимание, что это может быть использовано для группировки объектов через [page:.add]( object ) метод, который добавляет объект как ребенок,Однако лучше использовать [page:Group] для этого.

Конструктор

[name]()

Конструктор не принимает аргументов.

свойства

[property:Boolean castShadow]

Отображается ли объект в теневой карте. По умолчанию это *false*.

[property:Object3D children]

Массив с потомками объекта. См [page:Group] информация о группировке объектов вручную.

[property:Material customDepthMaterial]

Пользовательский материал глубины, который будет использоваться при рендеринге на карту глубины. Может использоваться только в контексте мешей. Когда отбрасывает тень с [page:DirectionalLight] или [page:SpotLight], если вы (a) измененяете положения вершины в вершинном шейдере, (b) с помощью displacement map, (c) используя alpha map с alphaTest, или (d) с помощью transparent texture с alphaTest, Вы должны указать customDepthMaterial для правильных теней. По умолчанию это *undefined*.

[property:Material customDistanceMaterial]

Такой же как [page:.customDepthMaterial customDepthMaterial], но используется с [page:PointLight]. По умолчанию это *undefined*.

[property:Boolean frustumCulled]

Когда это установлено, он проверяет каждый кадр, находится ли объект в усечённой области камеры, перед рендерингом объекта. В противном случае объект визуализируется каждый кадр, даже если он не виден. По умолчанию это *true*.

[property:Integer id]

readonly – Unique number for this object instance.

[property:Boolean isObject3D]

Используется для проверки того, являются ли этот или производные классы Object3Ds. По умолчанию это *true*.

Вы не должны менять это, так как он используется внутри для оптимизации.

[property:Layers layers]

Слой которому принадлежит объект. Объект виден, только если у него есть хотя бы один общий слой с [page:Camera] в использовании.

[property:Matrix4 matrix]

локалдьная матрица трансформации.

[property:Boolean matrixAutoUpdate]

Когда это установлено, он рассчитывает матрицу положения, (вращения или кватернионов) и масштаба каждый кадр, а также пересчитывает свойство matrixWorld. По умолчанию это [page:Object3D.DefaultMatrixAutoUpdate] (true).

[property:Matrix4 matrixWorld]

Глобальные трансформации объекта. Если Object3D не имеет родителя, то это идентично локальным трансформациям [page:.matrix].

[property:Boolean matrixWorldNeedsUpdate]

Когда это установлено, оно вычисляет matrixWorld в этом кадре и сбрасывает это свойство в false. По умолчанию это *false*.

[property:Matrix4 modelViewMatrix]

Это передается в шейдер и используется для расчета положения объекта.

[property:String name]

Необязательное название объекта (не должен быть уникальным). По умолчанию это пустая string.

[property:Matrix3 normalMatrix]

Это передается в шейдер и используется для расчета освещения для объекта. It is the transpose of the inverse of the upper left 3x3 sub-matrix modelViewMatrix этого объекта.

Причина этой специальной матрицы заключается в том, что просто используя modelViewMatrix может привести к ненулевой длине нормалей (при масштабировании) или в неперпендикулярном направлении (при неравномерном масштабировании).

С другой стороны, часть перевода modelViewMatrix не имеет отношения к вычислению нормалей.. Thus a Matrix3 is sufficient.

[property:Function onAfterRender]

Необязательный обратный вызов, который выполняется сразу после рендеринга Object3D. Эта функция вызывается со следующими параметрами: renderer, scene, camera, geometry, material, group.

[property:Function onBeforeRender]

Необязательный обратный вызов, который выполняется сразу перед рендерингом Object3D. Эта функция вызывается со следующими параметрами: renderer, scene, camera, geometry, material, group.

[property:Object3D parent]

Родитель объекта в [link:https://en.wikipedia.org/wiki/Scene_graph графе сцены]. Объект может иметь не более одного родителя.

[property:Vector3 position]

A [page:Vector3] представляющий локальную позицию объекта. По умолчанию это (0, 0, 0).

[property:Quaternion quaternion]

Локальное вращение объекта как [page:Quaternion Quaternion].

[property:Boolean receiveShadow]

Получает ли материал тени. По умолчанию это *false*.

[property:Number renderOrder]

Это значение позволяет порядок рендеринга по умолчанию [link:https://en.wikipedia.org/wiki/Scene_graph scene graph] objects быть переопределенным, хотя непрозрачные и прозрачные объекты остаются отсортированными независимо. Когда это свойство установлено для экземпляра [page:Group Group], все объекты потомков будут отсортированы и отображены вместе. Сортировка от низшей к высшей renderOrder. Значение по умолчанию *0*.

[property:Euler rotation]

Локальное вращение объекта (see [link:https://en.wikipedia.org/wiki/Euler_angles Euler angles]), в радианах.

[property:Vector3 scale]

Локальный масштаб объекта. По умолчанию это [page:Vector3]( 1, 1, 1 ).

[property:Vector3 up]

Это используется [page:.lookAt lookAt] method, например, определить ориентацию результата.
По умолчанию это [page:Object3D.DefaultUp] - that is, ( 0, 1, 0 ).

[property:Object userData]

Объект, который можно использовать для хранения пользовательских данных о Object3D. Он не должен содержать ссылки на функции, поскольку они не будут клонированы.

[property:String uuid]

[link:http://en.wikipedia.org/wiki/Universally_unique_identifier UUID] этого экземпляра объекта. Это присваивается автоматически, поэтому его не следует редактировать.

[property:Boolean visible]

Объект визуализируется, если *true*. По умолчанию это *true*.

Статические свойства

Статические свойства и методы определяются для каждого класса, а не для каждого экземпляра этого класса. Это означает, что изменение [page:Object3D.DefaultUp] или [page:Object3D.DefaultMatrixAutoUpdate] изменит значения [page:.up up] и [page:.matrixAutoUpdate matrixAutoUpdate] для каждого экземпляра Object3D (или производного класса) создано после внесения изменений (уже созданные Object3D не будут затронуты).

[property:Vector3 DefaultUp]

Направление по умолчанию объекта [page:.up up], также используется в качестве позиции по умолчанию для [page:DirectionalLight], [page:HemisphereLight] и [page:Spotlight] (который создает огни, сияющие сверху вниз).
Set to ( 0, 1, 0 ) by default.

[property:Boolean DefaultMatrixAutoUpdate]

The default setting for [page:.matrixAutoUpdate matrixAutoUpdate] for newly created Object3Ds.

Methods

[page:EventDispatcher EventDispatcher] methods are available on this class.

[method:this add]( [param:Object3D object], ... )

Добавляет * object * как потомок этого объекта. Может быть добавлено произвольное количество объектов. Любой текущий родительский объект переданного здесь объекта будет удален, поскольку у объекта может быть не более одного родительского объекта.

См [page:Group] информация о группировке объектов вручную.

[method:null applyMatrix]( [param:Matrix4 matrix] )

применяет матрицу трансформации к объекту и обновляет положение, вращение и масштаб.

[method:this applyQuaternion]( [param:Quaternion quaternion] )

Применяет вращение, представленное кватернионом к объекту.

[method:this attach]( [param:Object3D object] )

Добавляет * object * как потомок этого, сохраняя мировые преобразования объекта.

[method:Object3D clone]( [param:Boolean recursive] )

recursive -- если true, потомки объекта также клонируются. По умолчанию это true.

Возвращает клон этого объекта и опционально всех потомков.

[method:this copy]( [param:Object3D object], [param:Boolean recursive] )

recursive -- если true, потомки объекта тоже копируются. По умолчанию это true.

Скопирует переданный объект в объект this. Note:прослушиватели событий и пользовательские обратные вызовы ([page:.onAfterRender] и [page:.onBeforeRender]) не копируются.

[method:Object3D getObjectById]( [param:Integer id] )

id -- Уникальный номер экземпляра объекта

Выполняет поиск по дочерним объектам и возвращает первое с соответствующим id.
Обратите внимание, что идентификаторы назначаются в хронологическом порядке: 1, 2, 3, ..., увеличение на единицу для каждого нового объекта.

[method:Object3D getObjectByName]( [param:String name] )

name -- String соответствующий имени дочернего объекта Object3D.name.

Выполняет поиск по дочерним объектам и возвращает первый с совпадающим именем.
Обратите внимание, что для большинства объектов имя является пустой строкой по умолчанию. Вам придется установить его вручную, чтобы использовать этот метод.

[method:Object3D getObjectByProperty]( [param:String name], [param:Float value] )

name -- название свойства для поиска.
value -- значение этого свойства.

Выполняет поиск по дочерним объектам и возвращает первый со свойством, которое соответствует заданному значению.

[method:Vector3 getWorldPosition]( [param:Vector3 target] )

[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает вектор, представляющий положение объекта в мировом пространстве.

[method:Quaternion getWorldQuaternion]( [param:Quaternion target] )

[page:Quaternion target] — результат будет скопирован в этот Quaternion.

Возвращает кватернион, представляющий вращение объекта в мировом пространстве.

[method:Vector3 getWorldScale]( [param:Vector3 target] )

[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает вектор масштабных коэффициентов, примененных к объекту для каждой оси в мировом пространстве..

[method:Vector3 getWorldDirection]( [param:Vector3 target] )

[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает вектор, представляющий направление положительной оси Z объекта в мировом пространстве..

[method:Vector3 localToWorld]( [param:Vector3 vector] )

vector - Вектор, представляющий позицию в локальном (object) пространстве.

Преобразует вектор из локального пространства в мировое пространство.

[method:null lookAt]( [param:Vector3 vector] )
[method:null lookAt]( [param:Float x], [param:Float y], [param:Float z] )

vector - Вектор, представляющий положение в мировом пространстве.

Опциональные, [page:.x x], [page:.y y] и [page:.z z] компоненты в глобальном пространстве.

Поворот объекта на лицом(?) на точку в глобальном пространстве.

Этот метод не поддерживает объекты с неравномерно масштабированным родителем(-ями).

[method:Array raycast]( [param:Raycaster raycaster], [param:Array intersects] )

Абстрактный (пустой) метод для получения пересечений между приведенным лучом и этим объектом. Подклассы, такие как [page:Mesh], [page:Line], и [page:Points] реализуют этот метод в случае использования рэйкаста.

[method:this remove]( [param:Object3D object], ... )

Удаляет * объект * как дочерний элемент этого объекта. Произвольное количество объектов может быть удалено.

[method:this rotateOnAxis]( [param:Vector3 axis], [param:Float angle] )

axis -- Нормализованный вектор в пространстве объекта.
angle -- The угол в радианах.

Повернуть объект вдоль оси в локальном пространстве объекта. Ось предполагается нормализованной.

[method:this rotateOnWorldAxis]( [param:Vector3 axis], [param:Float angle] )

axis -- Нормализованный вектор в мировом пространстве.
angle -- Угол в радианах.

Повернуть объект вдоль оси в мировом пространстве. Ось предполагается нормализованной. Метод Предполагает, что нет повернутого родителя.

[method:this rotateX]( [param:Float rad] )

rad - угол поворота в радианах.

Вращает объект вокруг оси x в локальном пространстве.

[method:this rotateY]( [param:Float rad] )

rad - угол поворота в радианах.

Вращает объект вокруг y оси в локальном пространстве.

[method:this rotateZ]( [param:Float rad] )

rad - угол поворота в радианах.

Вращает объект вокруг z оси в локальном пространстве.

[method:null setRotationFromAxisAngle]( [param:Vector3 axis], [param:Float angle] )

axis -- Нормализованный вектор в пространстве объекта.
angle -- угол в радианах

Вызовы [page:Quaternion.setFromAxisAngle setFromAxisAngle]( [page:Float axis], [page:Float angle] ) в [page:.quaternion].

[method:null setRotationFromEuler]( [param:Euler euler] )

euler -- Угол Эйлера, определяющий величину вращения.
Calls [page:Quaternion.setRotationFromEuler setRotationFromEuler]( [page:Euler euler]) on the [page:.quaternion].

[method:null setRotationFromMatrix]( [param:Matrix4 m] )

m -- повернуть кватернион на компонент вращения матрицы.
Calls [page:Quaternion.setFromRotationMatrix setFromRotationMatrix]( [page:Matrix4 m]) on the [page:.quaternion].

Обратите внимание, что это предполагает, что верхние 3x3 из m является чистой матрицей вращения (т.е., немасштабированный).

[method:null setRotationFromQuaternion]( [param:Quaternion q] )

q -- normalized Quaternion.

Копируйет указанный кватернион в [page:.quaternion].

[method:Object toJSON]( [param:object meta] )

meta -- объект, содержащий метаданные, такие как materials, textures или images для объекта.
Преобразовать объект в three.js [link:https://github.com/mrdoob/three.js/wiki/JSON-Object-Scene-format-4 JSON Object/Scene format].

[method:this translateOnAxis]( [param:Vector3 axis], [param:Float distance] )

axis -- Нормализованный вектор в пространстве объекта.
distance -- Расстояние для переноса.

Translate объект на distance вдоль оси в пространстве объектов. Ось предполагается нормализованной.

[method:this translateX]( [param:Float distance] )

Переносит объект вдоль x оси в пространстве объекта на * distance * единиц.

[method:this translateY]( [param:Float distance] )

Переносит объект вдоль y оси в пространстве объекта на * distance * единиц.

[method:this translateZ]( [param:Float distance] )

Переносит объект вдоль z оси в пространстве объекта на * distance * единиц.

[method:null traverse]( [param:Function callback] )

callback - Функция с первым аргументом объекта object3D.

Выполняет обратный вызов для этого объекта и всех потомков.

[method:null traverseVisible]( [param:Function callback] )

callback - Функция с первым аргументом объекта object3D.

Как traverse, но обратный вызов будет выполняться только для видимых объектов. Потомки невидимых объектов не traverse.

[method:null traverseAncestors]( [param:Function callback] )

callback - Функция с первым аргументом объекта object3D.

Выполняет обратный вызов во всех предках.

[method:null updateMatrix]()

Обновляет локальные трансформации.

[method:null updateMatrixWorld]( [param:Boolean force] )

Обновляет глобальные трансформации объекта и его потомков.

[method:Vector3 worldToLocal]( [param:Vector3 vector] )

vector - мировой вектор.

Обновляет вектор из мирового пространства в локальное пространство.
