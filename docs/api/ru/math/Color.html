[name]

Класс, представляющий цвет.

Примеры

Color может быть инициализирован любым из следующих способов: //пустой конструктор - по умолчанию белый var color = new THREE.Color(); //Шестнадцатеричный цвет (рекомендуется) var color = new THREE.Color( 0xff0000 ); //RGB строка var color = new THREE.Color("rgb(255, 0, 0)"); var color = new THREE.Color("rgb(100%, 0%, 0%)"); //X11 color name - все 140 наименований цветов поддерживаются. //Note the lack of CamelCase in the name var color = new THREE.Color( 'skyblue' ); //HSL string var color = new THREE.Color("hsl(0, 100%, 50%)"); //Отдельные RGB значения между 0 и 1 var color = new THREE.Color( 1, 0, 0 );

Конструктор

[name]( [param:Color_Hex_or_String r], [param:Float g], [param:Float b] )

[page:Color_Hex_or_String r] - (опционально) если аргументы [page:Float g] и [page:Float b] объявлены, то red component of the color. Если они не определены, то это может быть [link:https://en.wikipedia.org/wiki/Web_colors#Hex_triplet hexadecimal triplet] (recommended), строка CSS-style string, или другой экземпляр Color.
[page:Float g] - (опционально) Если это определено, зеленый компонент цвета.
[page:Float b] - (опционально) Если это определено, синий компонент цвета.

Обратите внимание, что стандартный метод определения цвета в three.js это с [link:https://en.wikipedia.org/wiki/Web_colors#Hex_triplet шестнадцатеричный триплет], и этот метод используется в остальной части документации.

Когда все аргументы определены, тогда [page:Color_Hex_or_String r] это красный компонент, [page:Float g] это зеленый компонент и [page:Float b] синий компонент цвета.
ЕСли задан только [page:Color_Hex_or_String r] :

  • То это может быть [link:https://en.wikipedia.org/wiki/Web_colors#Hex_triplet hexadecimal triplet] представляющий цвет (recommended).
  • Это может быть другой Color экземпляр.
  • Это может быть CSS-style строка. Например:
    • 'rgb(250, 0,0)'
    • 'rgb(100%,0%,0%)'
    • 'hsl(0, 100%, 50%)'
    • '#ff0000'
    • '#f00'
    • 'red'

свойства

[property:Boolean isColor]

Используется для проверки того, являются ли этот или производные классы Colors. По умолчанию *true*.

Вы не должны менять это, так как он используется внутри для оптимизации.

[property:Float r]

Red channel value between 0 and 1. Default is 1.

[property:Float g]

Green channel value between 0 and 1. Default is 1.

[property:Float b]

Blue channel value between 0 and 1. Default is 1.

[method:Color lerp]( [param:Color color], [param:Float alpha] )

[page:Color color] - color to converge on.
[page:Float alpha] - interpolation factor in the closed interval [0, 1].

Linearly interpolates this color's RGB values toward the RGB values of the passed argument. The alpha argument can be thought of as the ratio between the two colors, where 0.0 is this color and 1.0 is the first argument.

[method:Color lerpHSL]( [param:Color color], [param:Float alpha] )

[page:Color color] - color to converge on.
[page:Float alpha] - interpolation factor in the closed interval [0, 1].

Linearly interpolates this color's HSL values toward the HSL values of the passed argument. It differs from the classic [page:.lerp] by not interpolating straight from one color to the other, but instead going through all the hues in between those two colors. The alpha argument can be thought of as the ratio between the two colors, where 0.0 is this color and 1.0 is the first argument.

[method:Color setStyle]( [param:String style] )

[page:String style] — color as a CSS-style string.

Sets this color from a CSS-style string. For example, "rgb(250, 0,0)", "rgb(100%, 0%, 0%)", "hsl(0, 100%, 50%)", "#ff0000", "#f00", or "red" ( or any [link:https://en.wikipedia.org/wiki/X11_color_names#Color_name_chart X11 color name] - all 140 color names are supported ).
Translucent colors such as "rgba(255, 0, 0, 0.5)" and "hsla(0, 100%, 50%, 0.5)" are also accepted, but the alpha-channel coordinate will be discarded.

Note that for X11 color names, multiple words such as Dark Orange become the string 'darkorange' (all lowercase).

[method:Color sub]( [param:Color color] )

Subtracts the RGB components of the given color from the RGB components of this color. If this results in a negative component, that component is set to zero.
