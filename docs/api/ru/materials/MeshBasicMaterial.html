[page:Material] →

[name]

Материал для отображения геометрии в простом затенении (flat или wireframe) .

Этот материал не зависит от света.

Constructor

[name]( [param:Object parameters] )

[page:Object parameters] - (optional) объект с одним или несколькими свойствами, определяющими внешний вид материала. Любое свойство материала (включая любые свойства, унаследованные от [page:Material]) могут быть переданы сюда.

Исключением является свойство [page:Hexadecimal color], которое может быть передано как шестнадцатеричная строка и по умолчанию это *0xffffff* (белый). [page:Color.set]( color ) вызывается внутренне.

свойства

См. общие свойства в базовом классе [page:Material].

[property:Texture alphaMap]

Альфа-карта - это текстура в оттенках серого, которая контролирует непрозрачность по всей поверхности. (black: полностью прозрачный; white: полностью непрозрачный). По умолчанию это null.

Используется только цвет текстуры, игнорируя альфа-канал, если он существует. Для RGB и RGBA текстур, [page:WebGLRenderer WebGL] рендер будет использовать зеленый канал при выборке этой текстуры из-за дополнительной точности, обеспечиваемой для зеленого в DXT-сжатии и без сжатия RGB 565 форматов. Luminance-only и luminance/alpha текстуры также будут работать как положено.

[property:Texture aoMap]

Красный канал этой текстуры используется в качестве карты окружающей окклюзии. По умолчанию это ноль. Карте aoMap требуется второй набор UVs, и, следовательно, будет игнорировать свойства текстуры [page:Texture repeat] и [page:Texture offset].

[property:Float aoMapIntensity]

Интенсивность эффекта ambient occlusion. По умолчанию это 1. Ноль означает отсутствие эффекта occlusion.

[property:Color color]

[page:Color] цвет материала по-умолчанию белый (0xffffff).

[property:Integer combine]

Как объединить результат цвета поверхности с картой окружающей среды, если есть.

Варианты [page:Materials THREE.Multiply] (default), [page:Materials THREE.MixOperation], [page:Materials THREE.AddOperation]. Если смесь выбрана, то [page:.reflectivity] используется для смешивания двух цветов.

[property:Boolean isMeshBasicMaterial]

Используется для проверки того, является ли этот или производные классы базовыми материалами сетки. По умолчанию это *true*.

Вы не должны изменять это, поскольку это используется внутренне для оптимизации.

[property:TextureCube envMap]

Карта окружения. По умолчанию это null.

[property:Texture lightMap]

light map. По умолчанию это null. The lightMap требуется второй набор UV координат, и, следовательно, будет игнорировать[page:Texture repeat] и [page:Texture offset] свойства текстуры.

[property:Float lightMapIntensity]

Интенсивность запеченного света. По умолчанию это 1.

[property:Boolean lights]

Влияет ли материал на свет. По умолчанию это *false*.

[property:Texture map]

color map. По умолчанию это null.

[property:Boolean morphTargets]

Определяет, использует ли материал morphTargets. По умолчанию это false.

[property:Float reflectivity]

Как карта окружающей среды влияет на поверхность; также см [page:.combine]. Значением по умолчанию является 1 и валидный диапазон между 0 (нет отражений) и 1 (полное отражение).

[property:Float refractionRatio]

ИНдекс преломления (IOR) воздуха (approximately 1) делится на показатель преломления материала. Используется с режимами отображения среды [page:Textures THREE.CubeRefractionMapping] и [page:Textures THREE.EquirectangularRefractionMapping]. Коэффициент преломления не должен превышать 1. По умолчанию это *0.98*.

[property:Boolean skinning]

Определите, использует ли материал скиннинг. По умолчанию это false.

[property:Texture specularMap]

Зеркальная карта, используемая материалом. По умолчанию это null.

[property:Boolean wireframe]

Визуализировать геометрию как каркас. По умолчанию это *false* (i.e. render as flat polygons).

[property:String wireframeLinecap]

Определить внешний вид концов линии. Возможные значения "butt", "round" и "square". По умолчанию это 'round'.

Это соответствует [link:https://developer.mozilla.org/en/docs/Web/API/CanvasRenderingContext2D/lineCap 2D Canvas lineCap] свойству, и оно игнорируется [page:WebGLRenderer WebGL] renderer.

[property:String wireframeLinejoin]

Определить внешний вид линейных соединений. Возможные значения "round", "bevel" и "miter". По умолчанию это 'round'.

This corresponds to the [link:https://developer.mozilla.org/en/docs/Web/API/CanvasRenderingContext2D/lineJoin 2D Canvas lineJoin] свойству, и оно игнорируется [page:WebGLRenderer WebGL] renderer.

[property:Float wireframeLinewidth]

Контролирует толщину каркаса. По умолчанию это 1.

Из-за ограничений [link:https://www.khronos.org/registry/OpenGL/specs/gl/glspec46.core.pdf OpenGL Core Profile] с [page:WebGLRenderer WebGL] renderer на большинстве платформ ширина линии всегда будет 1 независимо от установленного значения.
