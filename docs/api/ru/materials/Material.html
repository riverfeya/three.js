[name]

Абстрактный базовый класс для материалов.

Материалы описывают появление [page:Object объектов]. Они определены (в основном) независимым от визуализации способом, поэтому вам не нужно переписать материалы, если вы решили использовать другой рендер.

Следующие свойства и методы наследуются всеми другими типами материалов (хотя они могут иметь разные значения по умолчанию).

Constructor

[name]()

Это создает общий материал.

Свойства

[property:Float alphaTest]

Устанавливает альфа-значение, которое будет использоваться при запуске альфа-теста. Материал не будет визуализироваться, если непрозрачность ниже этого значения. По умолчанию *0*.

[property:Integer blendDst]

Смешивание назначения. По умолчанию это [page:CustomBlendingEquation OneMinusSrcAlphaFactor]. Смотрите факторы назначения [page:CustomBlendingEquation constants] для всех возможных значений.
Материалы [page:Constant blending] должен быть установлен в [page:Materials CustomBlending] чтобы это имело какой-либо эффект.

[property:Integer blendDstAlpha]

Прозрачность [page:.blendDst]. По умолчанию это *null*.

[property:Integer blendEquation]

Blending equation to use when applying blending. По умолчанию это [page:CustomBlendingEquation AddEquation]. See the blending equation [page:CustomBlendingEquation constants] для всех возможных значений.
Материалы [page:Constant blending] должен быть установлен в [page:Materials CustomBlending] чтобы это имело какой-либо эффект.

[property:Integer blendEquationAlpha]

The tranparency of the [page:.blendEquation]. По умолчанию это *null*.

[property:Blending blending]

Какое смешение использовать при отображении объектов с этим материалом.
This должен быть установлен в [page:Materials CustomBlending] to use custom [page:Constant blendSrc], [page:Constant blendDst] or [page:Constant blendEquation].
See the blending mode [page:Materials constants] для всех возможных значений. По умолчанию это [page:Materials NormalBlending].

[property:Integer blendSrc]

Смешивание источника. По умолчанию это [page:CustomBlendingEquation SrcAlphaFactor]. See the source factors [page:CustomBlendingEquation constants] для всех возможных значений.
Материалы [page:Constant blending] должен быть установлен в [page:Materials CustomBlending] for this to have any effect.

[property:Integer blendSrcAlpha]

The tranparency of the [page:.blendSrc]. По умолчанию это *null*.

[property:Boolean clipIntersection]

Changes the behavior of clipping planes so that only their intersection is clipped, rather than their union. По умолчанию это *false*.

[property:Array clippingPlanes]

Пользовательские плоскости отсечения, определенные как объект THREE.Plane в мировом пространстве. Эти плоскости относятся к объектам, к которым этот материал прикреплен. Точки в пространстве, чье расстояние до плоскости отрицательно, обрезаются (не отображаются). Это требует [page:WebGLRenderer.localClippingEnabled] to be *true*. See the [example:webgl_clipping_intersection WebGL / clipping /intersection] example. По умолчанию это *null*.

[property:Boolean clipShadows]

Определяет, следует ли обрезать тени в соответствии с clipping planes, указанными для этого материала. По умолчанию это *false*.

[property:Boolean colorWrite]

Whether to render материалы color. This can be used in conjunction with a mesh's [page:Integer renderOrder] property to create invisible objects that occlude other objects. По умолчанию это *true*.

[property:Object defines]

Custom defines to be injected into the shader. These are passed in form of an object literal, with key/value pairs. { MY_CUSTOM_DEFINE: '' , PI2: Math.PI * 2 }. The pairs are defined in both vertex and fragment shaders. По умолчанию это *undefined*.

[property:Integer depthFunc]

Which depth function to use. По умолчанию это [page:Materials LessEqualDepth]. See the depth mode [page:Materials constants] для всех возможных значений.

[property:Boolean depthTest]

Whether to have depth test enabled when rendering this material. По умолчанию это *true*.

[property:Boolean depthWrite]

Whether rendering this material has any effect on the depth buffer. По умолчанию это *true*.

When drawing 2D overlays it can be useful to disable the depth writing in order to layer several things together without creating z-index artifacts.

[property:Boolean stencilWrite]

Whether rendering this material has any effect on the stencil buffer. По умолчанию это *false*.

[property:Boolean stencilFunc]

The stencil comparison function to use. По умолчанию это [page:Materials AlwaysStencilFunc]. See stencil function [page:Materials constants] для всех возможных значений.

[property:Integer stencilRef]

The value to use when performing stencil comparisons or stencil operations. По умолчанию это *0*.

[property:Boolean stencilMask]

The bit mask to use when comparing against or writing to the stencil buffer. По умолчанию это *0xFF*.

[property:Integer stencilFail]

Which stencil operation to perform when the comparison function returns false. По умолчанию это [page:Materials KeepStencilOp]. See the stencil operations [page:Materials constants] для всех возможных значений.

[property:Boolean stencilZFail]

Which stencil operation to perform when the comparison function returns true but the depth test fails. По умолчанию это [page:Materials KeepStencilOp]. See the stencil operations [page:Materials constants] для всех возможных значений.

[property:Boolean stencilZPass]

Which stencil operation to perform when the comparison function returns true and the depth test passes. По умолчанию это [page:Materials KeepStencilOp]. See the stencil operations [page:Materials constants] для всех возможных значений.

[property:Boolean flatShading]

Определите, отображается ли материал с плоским затенением. По умолчанию это false.

[property:Boolean fog]

Whether the material is affected by fog. По умолчанию это *true*.

[property:Integer id]

Уникальный номер для этого экземпляра материала.

[property:Boolean isMaterial]

Used to check whether this or derived classes are materials. По умолчанию это *true*.

You should not change this, as it used internally for optimisation.

[property:Boolean lights]

Влияет ли материал на свет. По умолчанию это *true*.

[property:String name]

Необязательное название объекта (не обязано быть уникальным). По умолчанию это пустая строка.

[property:Boolean needsUpdate]

Указывает, что материал должен быть перекомпилирован.
Это свойство автоматически устанавливается на *true* при создании нового материала.

[property:Float opacity]

Плавающее значение в диапазоне *0.0* - *1.0* означает степень прозрачности. Где *0.0* указывает на полностью прозрачный, *1.0* полностью непрозрачный.
Если свойство материала [page:Boolean transparent] не задано в *true*, материал будет         оставаться полностью непрозрачными, и это значение влияет только на его цвет.
По умолчанию это *1.0*.

[property:Boolean polygonOffset]

Использовать ли смещение полигона. По умолчанию это *false*. Что соответствует *GL_POLYGON_OFFSET_FILL* WebGL функции.

[property:Integer polygonOffsetFactor]

Устанавливает коэффициент смещения полигона. По умолчанию это *0*.

[property:Integer polygonOffsetUnits]

Устанавливает единицы смещения полигона. По умолчанию это *0*.

[property:String precision]

Переопределить точность рендерера по умолчанию для этого материала. Может быть "*highp*", "*mediump*" или "*lowp*". По умолчанию это *null*.

[property:Boolean premultipliedAlpha]

Преумножать ли значение альфа (прозрачность). See [Example:webgl_materials_transparency WebGL / Materials / Transparency] для примера разницы. По умолчанию это *false*.

[property:Boolean dithering]

Следует ли применять дизеринг к цвету, чтобы устранить появление полос. По умолчанию это *false*.

[property:Integer shadowSide]

Определяет, с какой стороны фейсы отбрасывают тени. Когда установлено, может быть [page:Materials THREE.FrontSide], [page:Materials THREE.BackSide], или [page:Materials THREE.DoubleSide]. По умолчанию это *null*.
Если *null*, сторона отбрасывающая тени определяется следующим образом:

[page:Material.side] Боковые тени
THREE.FrontSide задняя сторона
THREE.BackSide Лицевая сторона
THREE.DoubleSide обе стороны

[property:Integer side]

Определяет, какая сторона граней будет отрисована - передняя, ​​задняя или обе. По умолчанию это [page:Materials THREE.FrontSide]. Other options are [page:Materials THREE.BackSide] and [page:Materials THREE.DoubleSide].

[property:Boolean toneMapped]

Определяет, является ли этот материал tone mapped соответствующим рендеру [page:WebGLRenderer.toneMapping toneMapping] setting. По умолчанию это *true*.

[property:Boolean transparent]

Определяет, является ли этот материал transparent. Это влияет на рендеринг, поскольку прозрачные объекты требуют специальной обработки и отображаются после непрозрачных объектов.
При значении true степень прозрачности материала         контролируется установкой его свойства [page:Float opacity] .
По умолчанию это *false*.

[property:String type]

Значение это строка 'Material'. Это свойство не нужно изменять, и оно может использоваться, чтобы найти все объекты этого типа в сцене.

[property:String uuid]

[link:http://en.wikipedia.org/wiki/Universally_unique_identifier UUID] этого экземпляра материала. Это автоматически присваивается, поэтому его не следует редактировать.

[property:Integer vertexColors]

Определяет, используется ли раскраска вершин. По умолчанию это [page:Materials THREE.NoColors]. Другие варианты [page:Materials THREE.VertexColors] и [page:Materials THREE.FaceColors].

[property:Boolean vertexTangents]

Определяет, используются ли предварительно вычисленные касательные вершин, которые должны быть предоставлены в атрибуте "tangent" vec4. При отключении касательные выводятся автоматически. Использование предварительно вычисленных касательных даст более точные детали карты нормалей в некоторых случаях, например, с зеркальными UV. По умолчанию это false.

[property:Boolean visible]

Определяет, является ли этот материал visible. По умолчанию это *true*.

[property:object userData]

Объект, который можно использовать для хранения пользовательских данных о материале. Он не должен содержать ссылки на функции, так как они не будут клонированы.

методы

[page:EventDispatcher EventDispatcher] методы доступны в этом классе.

[method:Material clone]( )

Вернуть новый материал с такими же параметрами, как у этого материала.

[method:Material copy]( [param:material material] )

Скопируйте параметры из переданного материала в этот материал..

[method:null dispose]()

Это удаляет материал. Текстуры материала не удаляются. Они должны быть утилизированы [page:Texture Texture].

[method:null onBeforeCompile]( [param:Shader shader], [param:WebGLRenderer renderer] )

Необязательный обратный вызов, который выполняется непосредственно перед компиляцией шейдерной программы. Эта функция вызывается с исходным кодом шейдера в качестве параметра. Полезно для модификации встроенных материалов.

[method:null setValues]( [param:object values] )

values -- a container with parameters.
Устанавливает свойства на основе *values*.

[method:Object toJSON]( [param:object meta] )

meta -- объект, содержащий метаданные, такие как текстуры или изображения для материала.
Преобразовать материал в формат three.js [link:https://github.com/mrdoob/three.js/wiki/JSON-Object-Scene-format-4 JSON Object/Scene format].
