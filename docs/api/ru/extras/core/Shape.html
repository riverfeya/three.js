[name]

Определяет произвольную плоскость 2D-формы, используя пути с необязательными отверстиями. Может использоваться с [page:ExtrudeGeometry], [page:ShapeGeometry], чтобы получить точки, или получить триангулированные грани.

var heartShape = new THREE.Shape(); heartShape.moveTo( 25, 25 ); heartShape.bezierCurveTo( 25, 25, 20, 0, 0, 0 ); heartShape.bezierCurveTo( 30, 0, 30, 35,30,35 ); heartShape.bezierCurveTo( 30, 55, 10, 77, 25, 95 ); heartShape.bezierCurveTo( 60, 77, 80, 55, 80, 35 ); heartShape.bezierCurveTo( 80, 35, 80, 0, 50, 0 ); heartShape.bezierCurveTo( 35, 0, 25, 25, 25, 25 ); var extrudeSettings = { amount: 8, bevelEnabled: true, bevelSegments: 2, steps: 2, bevelSize: 1, bevelThickness: 1 }; var geometry = new THREE.ExtrudeGeometry( heartShape, extrudeSettings ); var mesh = new THREE.Mesh( geometry, new THREE.MeshPhongMaterial() );

Примеры

Constructor

[name]( [param:Array points] )

points -- (optional) array of [page:Vector2 Vector2s].

Создает форму из точек. Первая точка определяет смещение, затем последовательные точки добавляются к [page:CurvePath.curves curves] массиву как [page:LineCurve LineCurves].

Если точки не указаны, создается пустая фигура и [page:.currentPoint] установлен на origin.

Properties

Смотри базовый класс [page:Path] для общих свойств.

[property:String uuid]

[property:array holes]

Массив [page:Path paths] которые определяют отверстия в форме.

Вызов [page:Curve.getPoints getPoints] в shape и [page:.holes] массив, и вернуть объект в форме: { shape holes } Где shape и holes это массивы [page:Vector2 Vector2s].

Получить массив [page:Vector2 Vector2s] которые представляют собой отверстия в форме.
