[page:Curve] → [page:CurvePath] →

[name]

2D представление пути. Класс предоставляет методы для создания контуров и путей 2D-фигур, аналогичных 2D Canvas API.

пример

var path = new THREE.Path(); path.lineTo( 0, 0.8 ); path.quadraticCurveTo( 0, 1, 0.2, 1 ); path.lineTo( 1, 1 ); var points = path.getPoints(); var geometry = new THREE.BufferGeometry().setFromPoints( points ); var material = new THREE.LineBasicMaterial( { color: 0xffffff } ); var line = new THREE.Line( geometry, material ); scene.add( line );

Constructor

[name]( [param:Array points] )

points -- (optional) массив [page:Vector2 Vector2s].

Создает Path из points. Первая точка определяет смещение, затем последовательные точки добавляются к [page:CurvePath.curves curves] массиву как [page:LineCurve LineCurves].

Если точки не указаны, создается пустой путь и [page:.currentPoint] устанавливается в origin.

свойства

Смотри базовый класс [page:CurvePath] для общих свойств.

[property:array currentPoint]

Текущее смещение пути. Любой новый [page:Curve] начнет добавляться здесь.

методы

Смотрите базовый [page:CurvePath] класс для общих методов.

[method:null absarc]( [param:Float x], [param:Float y], [param:Float radius], [param:Float startAngle], [param:Float endAngle], [param:Float clockwise] )

x, y -- Абсолютный центр дуги.
radius -- Радиус дуги.
startAngle -- Начальный угол в радианах.
endAngle -- Конечный угол в радианах.
clockwise -- Sweep дугу по часовой стрелке. По умолчанию *false*.

Добавляет абсолютное позиционирование [page:EllipseCurve EllipseCurve] к пути.

[method:null absellipse]( [param:Float x], [param:Float y], [param:Float xRadius], [param:Float yRadius], [param:Float startAngle], [param:Float endAngle], [param:Float clockwise], [param:Float rotation] )

x, y -- Абсолютный центр эллипса.
xRadius -- Радиус эллипса по x оси.
yRadius -- Радиус эллипса по y оси.
startAngle -- Начальный угол в радианах.
endAngle -- Конечный угол в радианах.
clockwise -- Sweep эллипс по часовой стрелке. По умолчанию false.
rotation -- Угол поворота эллипса в радианах, против часовой стрелки от положительной оси X. Необязательно, по умолчанию 0.

Добавляет абсолютное позиционирование [page:EllipseCurve EllipseCurve] к пути.

[method:null arc]( [param:Float x], [param:Float y], [param:Float radius], [param:Float startAngle], [param:Float endAngle], [param:Float clockwise] )

x, y -- Смещение центра дуги от последнего вызова.
radius -- Радиус дуги.
startAngle -- Начальный угол в радианах.
endAngle -- Конечный угол в радианах.
clockwise -- Sweep дуги по часовой стрелке. По умолчанию * false *.

Добавляет [page:EllipseCurve EllipseCurve] к пути, расположенный относительно [page:.currentPoint].

[method:null bezierCurveTo]( [param:Float cp1X], [param:Float cp1Y], [param:Float cp2X], [param:Float cp2Y], [param:Float x], [param:Float y] )

This creates a bezier curve from [page:.currentPoint] with (cp1X, cp1Y) and (cp2X, cp2Y) as control points and updates [page:.currentPoint] to x and y.

[method:null ellipse]( [param:Float x], [param:Float y], [param:Float xRadius], [param:Float yRadius], [param:Float startAngle], [param:Float endAngle], [param:Float clockwise], [param:Float rotation] )

x, y -- Смещение центра эллипса от последнего вызова.
xRadius -- Радиус эллипса в x axis.
yRadius -- Радиус эллипса в y axis.
startAngle -- Начальный угол в радианах.
endAngle -- Конечный угол в радианах.
clockwise -- Sweep эллипс по часовой стрелке. По умолчанию * false *.
rotation -- Угол поворота эллипса в радианах, против часовой стрелки от положительной оси X. Необязательно, по умолчанию * 0 *.

Добавляет[page:EllipseCurve EllipseCurve] к пути, расположен относительно [page:.currentPoint].

[method:null lineTo]( [param:Float x], [param:Float y] )

Подключает [page:LineCurve] из [page:.currentPoint] к X, Y пути.

[method:null moveTo]( [param:Float x], [param:Float y] )

Gthtvtoftn [page:.currentPoint] to x, y.

[method:null quadraticCurveTo]( [param:Float cpX], [param:Float cpY], [param:Float x], [param:Float y] )

Создает квадратичную кривую из [page:.currentPoint] c cpX b cpY как контрольная точка и обновляет [page:.currentPoint] в x и y.

[method:null setFromPoints]( [param:Array vector2s] )

points -- массив [page:Vector2 Vector2s].

Points добавляются в [page:CurvePath.curves curves] массив как [page:LineCurve LineCurves].

[method:null splineThru] ( [param:Array points] )

points - массив [page:Vector2 Vector2s]

Подключает новый [page:SplineCurve] к пути.
