[name]

Класс, содержащий вспомогательные функции для фигур.

Обратите внимание, что все это линейные функции, поэтому необходимо рассчитать отдельно для x, y (и z, w если есть) компонентов вектора.

методы

[method:Number area]( contour )

contour -- 2D polygon. An array of THREE.Vector2()

Рассчитать площадь ( 2D ) contour polygon.

[method:Boolean isClockwise]( pts )

pts -- точки, определяющие 2D polygon

Обратите внимание, что это линейная функция, поэтому необходимо отдельно вычислять компоненты x, y многоугольника..

Используется внутри [page:Path Path], [page:ExtrudeGeometry ExtrudeGeometry] и [page:ShapeGeometry ShapeGeometry].

[method:Array triangulateShape]( contour, holes )

contour -- 2D polygon.
holes -- массив отверстий

Используется внутри [page:ExtrudeGeometry ExtrudeGeometry] и [page:ShapeGeometry ShapeGeometry] рассчитать грани в фигурах с отверстиями.
