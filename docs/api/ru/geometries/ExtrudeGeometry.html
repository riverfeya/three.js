[name]

Создает экструзионную геометрию из формы контура.

пример

var length = 12, width = 8; var shape = new THREE.Shape(); shape.moveTo( 0,0 ); shape.lineTo( 0, width ); shape.lineTo( length, width ); shape.lineTo( length, 0 ); shape.lineTo( 0, 0 ); var extrudeSettings = { steps: 2, depth: 16, bevelEnabled: true, bevelThickness: 1, bevelSize: 1, bevelOffset: 0, bevelSegments: 1 }; var geometry = new THREE.ExtrudeGeometry( shape, extrudeSettings ); var material = new THREE.MeshBasicMaterial( { color: 0x00ff00 } ); var mesh = new THREE.Mesh( geometry, material ) ; scene.add( mesh );

Конструктор

[name]([param:Array shapes], [param:Object options])

shapes — Shape or an array of shapes.
options — Object that can contain the following parameters.

  • curveSegments — int. Количество точек на кривых. По умолчанию это 12.
  • steps — int. Количество точек, используемых для разделения сегментов по глубине выдавленного сплайна. По умолчанию 1.
  • depth — float. Глубина выдавливания формы. По умолчанию 100.
  • bevelEnabled — bool. Применить фаски к форме. По умолчанию true.
  • bevelThickness — float. Насколько глубоко в первоначальную форму уходит скос. По умолчанию 6.
  • bevelSize — float. Расстояние от контура формы, на которой проходит скос. По умолчанию bevelThickness - 2.
  • bevelOffset — float. Расстояние от контура формы, с которой начинается скос. По умолчанию 0.
  • bevelSegments — int. Количество слоев скоса. По умолчанию 3.
  • extrudePath — THREE.Curve. A 3D spline path вдоль которого форма должна быть выдавлена.
  • UVGenerator — Object. объект, который обеспечивает UV функции генератора

Этот объект выдавливает 2D форма в 3D геометрия.

При создании сетки с этой геометрией, если вы хотите использовать отдельный материал для ее поверхности и выдавленных сторон, вы можете использовать массив материалов. Первый материал будет нанесен на лицо; второй материал будет нанесен по сторонам.
