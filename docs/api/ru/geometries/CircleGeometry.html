[name]

CircleGeometry это простая форма Евклидовой геометрия. Он состоит из ряда треугольных сегментов, которые ориентированы вокруг центральной точки и простираются до заданного радиуса. Он построен против часовой стрелки от начального угла и заданного центрального угла. Его также можно использовать для создания правильных многоугольников, где количество сегментов определяет количество сторон.

Пример

var geometry = new THREE.CircleGeometry( 5, 32 ); var material = new THREE.MeshBasicMaterial( { color: 0xffff00 } ); var circle = new THREE.Mesh( geometry, material ); scene.add( circle );

Конструктор

[name]([param:Float radius], [param:Integer segments], [param:Float thetaStart], [param:Float thetaLength])

radius — радиус круга, по-умолчанию = 1.
segments — Количество сегментов (triangles), минимум = 3, дефолт = 8.
thetaStart — Начальный угол для первого сегмента, default = 0 (положение на три часа).
thetaLength — Центральный угол, часто называемый тета, кругового сектора. По умолчанию 2*Pi, что делает полный круг.
