Список устаревших API

Поскольку Three.js имеет быстро развивающийся интерфейс, вы можете встретить примеры, которые предлагают использовать элементы API, которые больше не являются частью ядра.

Ниже приведен список таких элементов, а также информация об их заменах..

Audio

[page:Audio]

Audio.load устарел. Используйте [page:AudioLoader] instead.

[page:AudioAnalyser]

AudioAnalyser.getData() был переименован в [page:AudioAnalyser.getFrequencyData]().

[page:BinaryTextureLoader]

BinaryTextureLoader был переименован в [page:DataTextureLoader].
